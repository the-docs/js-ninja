## 5.6 즉시실행함수(Immediate function)

```javascript
(function(){})()
```

```javascript
(...)()
```

```javascript
var someFunction = function() { ... };
result = someFunction();
```


```javascript
var someFunction = function() { ... };
result = (someFunction)();
```

```javascript
(function(){...})();
```

```javascript
(function(){
  statement - 1;
  statement - 2;
  statement - n;
})();
```
