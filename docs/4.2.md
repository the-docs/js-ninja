## 4.2 재귀 함수

```javascript
function isPalindrome(text) {
  if (text.length < 1) {
    return true;
  }
  if (text.charAt(0) != text.charAt(text.length - 1)) {
    return false;
  }
  return isPalindrome(text.substr(1, text.length - 2));
}

assert(isPalindrome('tomot') === true,
  'tomot는 팰린드롬이 맞음.');
assert(isPalindrome('apple') === false,
  'apple는 팰린드롬이 아님.');
```
